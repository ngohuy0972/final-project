## Features 
Guest :
1. View Product
2. Register
3. Add product to cart
4. Customer

Login :
1. Add product to cart
2. Checkout (No payment gateway implemented, all checkout status=PAID)
3. View purchase history
4. Admin

Login to admin panel :
1. CRUD product
2. CRUD size
3. Manage order
4. View user
5. Notes
